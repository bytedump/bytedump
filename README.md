## 🚀 Featured Projects

### 📊 [zabbix-printer-monitoring](https://github.com/bytedump/zabbix-printer-monitoring)
A Dockerized **Zabbix + Grafana** stack for centrally monitoring toner levels across a network printer fleet. Zabbix collects data over SNMP using the standard `Printer-MIB`, and a Python script consumes both the Zabbix and Grafana APIs to **auto-generate a dashboard** in a TV-friendly kiosk layout.

`Python` · `Docker` · `Zabbix` · `Grafana` · `SNMP` · `PostgreSQL`

### 🖥️ [windows-automatization](https://github.com/bytedump/windows-automatization)
Hands-free **Windows 11 provisioning** straight from a USB drive: an `autounattend.xml` for unattended install plus a **PowerShell GUI** to configure user, network, printers, apps, and Outlook signature. Secrets stay off the repo through templates and `.gitignore`.

`PowerShell` · `autounattend.xml` · `Windows 11`
